# Adhikar Setu AI

Adhikar Setu AI is a mobile-first, multilingual legal awareness platform designed to democratize access to legal information in India.

The system simplifies complex legal rights into easy-to-understand language and provides personalized, step-by-step complaint guidance. It integrates AWS Bedrock for AI processing, Amazon Transcribe for speech-to-text, Amazon Polly for text-to-speech, and AWS Lambda for scalable backend operations.

## Key Features
- AI-powered legal simplification
- Voice-enabled interaction
- Multilingual support
- Scam Risk Alert Mode
- Low-Literacy Explanation Mode
- Serverless AWS architecture
